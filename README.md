## 🚀 How to Use

1. Download the `XboxGamePassUnlocker.exe` from the latest release.
2. **Completely close** the Xbox app (including from the system tray).
3. Right-click the `.exe` file and select **"Run as administrator"**.
4. Wait a few seconds while the tool does its work.
5. Open the Xbox app again — all Game Pass titles should now appear as available to download/play.

**Tip:** If it doesn’t work the first time, restart your PC and try again.

## ⚡ Common Issues

- **Tool doesn’t work** → Make sure you ran it as administrator and fully closed the Xbox app.
- **Antivirus blocks the file** → This is common with unlockers (false positive). Add an exception if needed.
- **Games go back to requiring subscription** → You may need to run the tool again after restarting the Xbox app or PC.
